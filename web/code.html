SOBRE OS FRAGMENTOS

COMO CONSEGUIR FRAGMENTOS

Existem 2 formas de adquirir os fragmentos, a primeira delas é pelo comando "ash adfly" onde se resgata o código usando o comando "ash adlink <code>", outra forma seria o servidor tendo VIP GUILD, onde a cada comando usado na ASHLEY dentro daquele servidor você tem a chance de dropar um fragmento.

O VIP GUILD da ASHLEY custa R$ 30,00 e dura um total de 30 DIAS, também existe uma forma alternativa de conseguir o mesmo gratuitamente, preenchendo completamento a lateral esquerda do comando "ash perfil" usando o comando "ash rifa" para isso, assim você consegue ate 20 Blesseds Ethernyas, faltando apenas 10, para conseguir seu vip de guilda.

QUAL O VALOR DOS FRAGMENTOS

Os fragmentos são frações da Blessed Ethernya, uma pedra lendária que foi usada por muito tempo como cunho dos mais altos reis de todas as províncias de Sonard. Uma unidade deste item em seu inventário é o equivalente a R$ 1,00.

Para obter uma Blessed Ethernya completa vocè irá precisar de 1.000 fragmentos da mesma. Obtendo-as com o VIP GUILD ou com o ASH ADFLY.
